Introduction

Welcome to our site.
This site has a list of all the new airing/streaming anime in Japan, and which fansub groups are subbing (or plan to sub) them.
Anyone may edit, but you must sign up first on GitHub or send us a mail.
We are only adapting fansubDB website into others language. If you don't see yours, just do a Pull Request ;)

Can I help you?

Of course !
Don't hesitate to improve it, host it, modify it (and please inform me).

You can add your own language and fansub: just follow the french "fr" version, and e-mail me or open an issue on GitHub.

My repo is on https://github.com/fansubdb/fansubdb.github.io.
